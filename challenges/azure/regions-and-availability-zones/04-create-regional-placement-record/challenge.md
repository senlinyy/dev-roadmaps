Create the first production resource group for `devpolaris-orders-api` in the region the team chose. The resource group name is `rg-devpolaris-orders-prod-eastus`, the location is `eastus`, and the required tags are `service=orders-api`, `env=prod`, and `region-plan=primary`.

Your job:

1. **Create the resource group** named `rg-devpolaris-orders-prod-eastus`.
2. **Place it in** `eastus`.
3. **Apply the required tags** so the group shows what service, environment, and regional plan it belongs to.

The grader checks Azure resource group state, not a prose explanation.
