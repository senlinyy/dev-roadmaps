Use Azure CLI evidence for the production order database. The SQL server is `sql-devpolaris-orders-prod` in `rg-devpolaris-data-prod`. The database is `sqldb-devpolaris-orders-prod`.

Your job:

1. **Inspect** the SQL server and its public network setting.
2. **Inspect** the database status, tier, and size.
3. **Inspect** the server firewall rules so the team can see whether public access is being opened.

The grader checks for server, database, private access, online status, and firewall evidence.
