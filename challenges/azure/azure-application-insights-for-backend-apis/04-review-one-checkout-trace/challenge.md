A release review asks for the full Application Insights story for operation ID `checkout-5001`. Use component `appi-devpolaris-orders-prod`.

Your job:

1. **Query** the request result.
2. **Query** dependencies for the same operation.
3. **Query** exceptions for the same operation.
4. **Query** traces for the same operation.

The grader checks that your output connects request, dependency, exception, trace, and correlation evidence.
