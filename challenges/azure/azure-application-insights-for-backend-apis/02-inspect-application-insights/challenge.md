Use Azure evidence for Application Insights component `appi-devpolaris-orders-prod` in `rg-devpolaris-observability-prod`. This component observes `devpolaris-orders-api`.

Your job:

1. **Inspect** the component and workspace connection.
2. **Query** recent request evidence.
3. **Query** dependency evidence for backend calls.

The grader checks that the component, request stream, and dependency stream are available.
