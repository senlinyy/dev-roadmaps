Azure gives every resource a full ID that shows the subscription path and resource type. Inspect `rg-devpolaris-orders-staging` and use output that includes the full resource ID.

Your job:

1. **Inspect the resource group** named `rg-devpolaris-orders-staging`.
2. **Confirm the subscription path** includes `sub-devpolaris-training`.
3. **Confirm the full resource ID** ends with `resourceGroups/rg-devpolaris-orders-staging`.
4. **Confirm the tags** still identify the orders staging API.

The grader checks Azure CLI evidence for the resource ID, name, location, and tags.
