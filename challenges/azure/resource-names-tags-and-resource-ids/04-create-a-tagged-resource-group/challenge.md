Create the staging resource group for the orders API inventory. The resource group name is `rg-devpolaris-orders-staging`, the location is `eastus`, and the required tags are `team=orders`, `env=staging`, and `service=orders-api`.

Your job:

1. **Create the resource group** named `rg-devpolaris-orders-staging`.
2. **Place it in** `eastus`.
3. **Apply the required tags** for team, environment, and service.

The grader checks Azure resource group state, not a prose explanation.
