Before changing AWS resources, a teammate asks you to prove which account and Region this terminal is using. Treat the CLI as another doorway into the same AWS APIs the Console uses.

Your job:

1. **Ask AWS who you are** so you can see the current account.
2. **Check the configured Region** before assuming where resources will appear.
3. **Leave the evidence in the terminal output** so the grader can see both values.

The grader checks the command output, not a prose explanation.
