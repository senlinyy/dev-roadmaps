The orders team needs a small S3 bucket for exported order files. Create it with a name that says what it belongs to, then verify the bucket location.

Your job:

1. **Create an S3 bucket** named `devpolaris-orders-exports-prod` in `us-east-1`.
2. **Check the bucket location** after creation.
3. **Keep the command output visible** so the grader can see the verification.

The grader checks the simulated AWS resource state and the verification output.
