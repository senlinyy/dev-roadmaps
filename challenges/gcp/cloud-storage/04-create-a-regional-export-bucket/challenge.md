Create the production export bucket for the Orders analytics handoff. The bucket name is `devpolaris-orders-exports-prod`, the project is `dp-orders-prod`, the location is `us-central1`, and the labels are `team=orders`, `env=prod`, and `purpose=exports`.

Your job:

1. **Set the active project** to `dp-orders-prod`.
2. **Create the bucket** in `us-central1`.
3. **Apply the ownership labels** during creation.

The grader checks simulated Google Cloud bucket state, not a prose explanation.
