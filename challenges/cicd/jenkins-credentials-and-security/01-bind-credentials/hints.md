`withCredentials` takes an array of binding helpers and a closure. For AWS keys, use the `[$class: 'AmazonWebServicesCredentialsBinding', credentialsId: 'devpolaris-aws-deploy', accessKeyVariable: 'AWS_ACCESS_KEY_ID', secretKeyVariable: 'AWS_SECRET_ACCESS_KEY']` form, or the simpler `usernamePassword(...)` if you mapped the keys that way. For the Slack webhook, the `string(credentialsId: 'devpolaris-slack-webhook', variable: 'SLACK_WEBHOOK')` binding is the right shape. Both bindings live in the same array and the inner `sh` references the variables exactly as before.
