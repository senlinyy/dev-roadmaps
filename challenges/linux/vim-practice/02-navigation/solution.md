All navigation happens in Normal mode (no need to press `i`):

- `gg` jumps to line 1, `G` jumps to the last line
- `:15` jumps to line 15
- `w` moves forward one word, `b` moves backward one word
- `0` jumps to column 1, `$` jumps to end of line
- `Ctrl+d` scrolls half-page down, `Ctrl+u` scrolls half-page up

Quit without saving with `:q` since you are just reading.
