sectionSlug: reading-ls--l-output
order: 1
---

A server has several important files and directories. Your job is to inspect their permissions using `ls -l` and identify what access each file allows.

1. Run `ls -l /etc` to see the permissions of files in the `/etc` directory.
2. Run `ls -l /opt/deploy` to inspect the deployment directory contents.
3. Use `cat /home/dev/report.txt` to read the report file that summarizes what you found.
