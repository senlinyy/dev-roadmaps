A shell script is just a text file containing commands. Let's create one, make it executable, and run it.

You start in `/home/dev`. Your job:

1. **Create a script** called `hello.sh` that prints `Hello from DevPolaris!` (use `vim` to write the file, or `echo "..." > hello.sh` if you prefer).
2. **Make it executable** with `chmod`.
3. **Run it directly** with `./hello.sh`.

The grader checks that you used `chmod` and `./hello.sh`, and that `Hello from DevPolaris!` appears in the output.
