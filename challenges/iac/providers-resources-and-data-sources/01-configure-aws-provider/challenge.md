Complete the AWS provider setup in `main.tf`.

Requirements:

1. **Required provider source:** `hashicorp/aws`.
2. **Version constraint:** `~> 5.0`.
3. **Provider region:** `eu-west-2`.
4. **Default tags:** `project = "devpolaris-orders"`, `managed_by = "terraform"`.
