This project already has the Cargo shape from the article. Make the binary print a clear message from main.

Your job:

1. **Open** src/main.rs.
2. **Change** the printed message to hello from cargo.
3. **Run** the check; the grader executes cargo run and inspects stdout.

`Cargo.toml` is read-only project setup; edit `src/main.rs` only.

The grader checks the program output, not a prose explanation.
