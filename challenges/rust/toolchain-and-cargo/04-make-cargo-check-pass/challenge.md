Use cargo check as the quick compile-feedback command. The library has a classic expression mistake.

Your job:

1. **Inspect** src/lib.rs.
2. **Fix** the function so it returns the computed value.
3. **Run** the check; the grader executes cargo check.

`Cargo.toml` is read-only project setup; edit `src/lib.rs` only.

The grader checks that the project type-checks successfully.
