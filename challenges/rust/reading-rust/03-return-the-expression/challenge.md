The function body computes the right value, but the final semicolon changes what the block returns.

Your job:

1. **Inspect** the final line of score.
2. **Make** the function return the computed integer `15`.
3. **Run** the tests.

`Cargo.toml` is read-only project setup. The read-only tests live in `tests/score_test.rs`; edit `src/lib.rs` only.

The grader checks the function return value.
