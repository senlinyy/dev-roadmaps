Implement the small function shape from the article: borrowed text in, count out.

Your job:

1. **Implement** word_count(text: &str) -> usize.
2. **Use** the standard whitespace splitting behavior.
3. **Run** the tests.

`Cargo.toml` is read-only project setup. The read-only tests live in `tests/word_count_test.rs`; edit `src/lib.rs` only.

The grader checks several strings, including empty and extra-spaced input.
